I've setup a logic analyzer using Arduino before and the most glaring
issue was the overhead of the arduino kernel and lack of buffer. The
project included herein solves these issues to give a cheap quick
16Mhz logic analyzer.

Source:
http://www.uchobby.com/index.php/2008/09/09/avr-logic-analyzer/
